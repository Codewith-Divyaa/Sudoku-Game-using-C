# Sudoku Game in C 🎮

A console-based Sudoku game developed using C language.

## Features

* Login System
* Sudoku Gameplay
* Score Tracking
* File Handling
* Dynamic Memory Allocation
* Modular Programming

## Technologies Used

* C Language
* Code::Blocks IDE
* GCC Compiler

## Project Modules

* main.c
* login.c
* sudoku.c
* record.c

## Concepts Used

* Structures
* Pointers
* Arrays
* File Handling
* Dynamic Memory Allocation
* Functions
